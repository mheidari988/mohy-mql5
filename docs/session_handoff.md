# MOHY Session Handoff

Use this short format at the end of completed tasks:

- Scope:
- Docs:
- Code:
- Verification:
- Next:

Rules:
- Keep each line concrete and brief.
- Say `none` for docs or code when unchanged.
- Name commands that were run and whether they passed.
- If blocked, replace `Scope` with `Current state` and put the blocker in `Next`.
